# juneeForFinger
Situs tes mengetik sederhana Bahasa Indonesia yang bisa dihost di GitHub Pages.

## Cara pakai (lokal)
1. Simpan `index.html`, `styles.css`, dan `script.js` di folder yang sama.
2. Buka `index.html` di browser.

## Deploy ke GitHub Pages
1. Buat repo baru di GitHub (misal `juneeForFinger`).
2. Upload ketiga file (`index.html`, `styles.css`, `script.js`).
3. Masuk ke **Settings → Pages**.
4. Pilih branch `main` dan folder `/ (root)`, lalu klik **Save**.
5. Tunggu 1–2 menit. Situsmu akan aktif di `https://<username>.github.io/juneeForFinger/`.
